LGU Document Management System

This project was developed as our undergraduate capstone project for the Municipality of Panay. It is a web-based Document Management System designed to improve document organization, tracking, approval, and retrieval across different municipal offices.

The system serves as a digital solution that replaces manual document handling, making government transactions faster, more organized, and easier to monitor.

This case study presents the project's objectives, system design, implementation process, and documentation.
